## Onde baixar

A extensão está disponível na **Chrome Web Store** (link fornecido após signup
no painel "Extensão Chrome → Instalar").

⚠️ Se a Chrome Web Store ainda não está liberada (instância nova),
o link na Chrome Web Store será adicionado quando a publicação for aprovada
pelo Google. Use a instalação manual abaixo como alternativa.

## Instalação via Chrome Web Store

1. Clique no link no painel
2. "Adicionar ao Chrome"
3. "Adicionar extensão"
4. Pronto — ícone roxo aparece no canto superior direito

## Instalação manual (modo desenvolvedor)

Se você recebeu um arquivo `.zip` da extensão:

1. Descompacte em uma pasta permanente (NÃO em Downloads)
2. Abra `chrome://extensions/`
3. Ative "Modo desenvolvedor" (canto superior direito)
4. Clique "Carregar sem compactação"
5. Selecione a pasta descompactada

## Conectando ao painel

1. Clique no ícone da extensão (roxo)
2. Clique "Conectar ao painel"
3. Faça login (mesmas credenciais do site)
4. Pronto — extensão sincronizada

## Verificando que funcionou

1. Abra `web.whatsapp.com`
2. Aguarde QR code aparecer (se primeira vez) e escaneie
3. Após carregar, deve aparecer um painel lateral roxo do WhatsHybrid

Se NÃO aparecer:
- Verifique que a extensão está habilitada em `chrome://extensions/`
- Recarregue a página do WhatsApp Web
- Vê os logs do console (F12) por erros que começam com `[WHL]`
