## O que é

A "Base de conhecimento" é texto livre onde você cola informações da empresa.
A IA usa esse texto via RAG (Retrieval-Augmented Generation) para responder.

## O que colocar

✅ **DEVE colocar:**
- Horário de funcionamento
- Endereço(s)
- Regras de troca / devolução
- Política de frete
- FAQ recorrente
- Catálogo simplificado (se aplicável)
- Formas de pagamento
- Tempo médio de entrega

❌ **NÃO coloque:**
- Senhas / dados internos
- Informação que muda toda hora (preços instáveis)
- Dados pessoais de funcionários

## Tamanho recomendado

- **Mínimo:** 500 caracteres (senão IA fica genérica demais)
- **Máximo:** 50.000 caracteres
- **Sweet spot:** 2.000-5.000 caracteres

## Atualização

Sempre que mudar política da empresa, atualize aqui. A mudança é instantânea
(próximas mensagens já usam a nova base).
